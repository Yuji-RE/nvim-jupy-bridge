# Changelog

## [0.0.9] - 2025-01-05

### Added
- Neovim の `:w` をトリガに、VS Code 側で **対象セルへジャンプ → 実行** を自動化。
- Neovim から **Run All Cells / Run Cells Below** を実行可能。
- `NvimJupy Debug` による性能内訳ログ（保存→検知→ジャンプ→実行の所要時間）を追加。

### Changed
- ワークフローを **双方向運用**（VS Code 側の編集も反映可能）に拡張し、柔軟性を向上。
- 遅延制御を見直し、固定遅延を短縮しても安定動作するよう改善。

---

## [0.0.1] - 2025-11-16

### Added
- Neovim で編集する `.py` と、VS Code で開く `.ipynb` の連携を前提に、基本ワークフローを成立。
- VS Code 側の **アクティブセル実行（フォーカス依存）** による最小プロトタイプを実装。
